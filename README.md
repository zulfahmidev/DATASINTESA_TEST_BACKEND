# Backend Technical Test Datasintesa - Zulfahmi

Hasil teknikal tes sebagai Fullstack Developer di perusahaan Datasintesa oleh Zulfahmi.

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

1. Pertama, install proyek dengan perintah berikut:

    ```bash
    npm install
    ```
2. Buat file `.env` dan isikan seperti dibawah atau sesuaikan:

    ```properties
    DB_HOST = 'localhost'
    DB_PORT = '27017'
    DB_NAME = 'datasintesa'
    ```
3. Jalankan proyek dengan perintah berikut:

    ```bash
    npm run start
    ```
